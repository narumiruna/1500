臺灣臺北地方法院 113 年度金訴字第 51 號案件資料庫

專案簡介

本專案旨在建立一個公開、可追溯的案件資料庫，蒐集並保存臺灣臺北地方法院 113 年度金訴字第 51 號（被告柯文哲等人違反貪污治罪條例等案件）相關之法院新聞稿、庭審錄影與語音逐字稿等資料。

專案名稱「1500」源自起訴書中被告被訴收受 1,500 萬元賄賂之指控；法院審理後認定該部分除被告自承之書面陳述外，無其他補強證據佐證，故未認定此部分犯罪事實。

資料處理流程

  1. 擷取：從臺北地方法院官網下載判決新聞稿，並轉換為 Markdown 格式保存。
  2. 下載：使用 yt-dlp 從 YouTube 批次下載法院公開之庭審錄影。
  3. 轉錄：使用 OpenAI Whisper 語音辨識模型，將庭審錄影轉為文字逐字稿。
  4. 存檔：所有資料以 Git 版本控制進行管理，確保完整性與可追溯性。
