# TASTE

- For audio download workflows, prefer original source audio format (no transcoding) over MP3 conversion unless explicitly requested.
- For Python scripts under `scripts/`, manage per-script dependencies using `uv add --script` and keep script metadata blocks up to date.
- For CLI design, prefer named options over positional arguments when both are reasonable.
- For any change under `scripts/`, update the related documentation in the same turn.
- Keep top-level site navigation aligned across pages; when a new primary content section is added (e.g., judgement content), include it as a dedicated top tab.
